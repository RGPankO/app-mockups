Frequently Asked Questions

Which program should I start with?

If you're new to pelvic floor exercises, start with the Beginner program (4 weeks). Postpartum moms can use the Recovery program. The Advanced program is for those who've completed Beginner.

How long are the exercises?

Most sessions are 5-10 minutes. The app guides you through each exercise with a timer, so you know exactly when to squeeze and relax.

Does the app work offline?

Yes! FloorStrong works 100% offline. All exercises and programs are built into the app.

Is this medical advice?

No. FloorStrong provides general exercise guidance. If you have pelvic floor dysfunction, prolapse, or are recovering from childbirth, please consult a pelvic floor physical therapist.

Can men use this app?

Pelvic floor health matters for everyone! While designed for women, the exercises are beneficial for all genders.
