Feature Breakdown

Top bar + brand card

UI element: Logo, name, supportive pill, trial badge. Why: builds trust and signals community‑first tone. Value: makes the app feel welcoming vs clinical. Real estate: first touchpoint for identity and pricing clarity.

Dashboard overview

UI element: Today’s overview card with cycle day and irregular range. Why: validates irregular cycles (35–90+). Value: immediate context without judgement. Real estate: most‑used info should be above the fold.

Symptom snapshot grid

UI element: Four quick tiles for mood, cravings, energy, bloat. Why: fast reflection without heavy logging. Value: supports emotional check‑ins. Real estate: replaces sterile charts with human signals.

Consistency score

UI element: Progress bar with weekly logging. Why: encourages gentle habit‑building. Value: highlights progress even if cycles are long. Real estate: creates a simple, motivating win.

Track screen toggles

UI element: Symptom chips for acne, hair growth, weight, mood, cravings, fatigue. Why: one‑tap logging. Value: reduces burden and keeps data consistent. Real estate: core behavior of the product.

Intensity scores

UI element: Numeric intensity cards. Why: makes subjective symptoms trackable. Value: fuels AI insights and doctor prep. Real estate: bridges feelings to actionable data.

Insights chart

UI element: Cycle variability line with trend note. Why: irregular cycles need pattern tracking. Value: helps users feel less “random.” Real estate: visual proof of progress.

Correlation cards

UI element: AI‑detected links (sleep → cravings, hair growth → mood). Why: transforms logs into insights. Value: personalizes small changes. Real estate: highlights why the app matters.

Community feed

UI element: Thread list with Q&A and success stories. Why: PCOS can be isolating. Value: peer proof and emotional support. Real estate: emphasizes belonging over diagnosis.

Moderated safety signals

UI element: “Verified peers” label and professional notes. Why: avoids misinformation. Value: builds trust for health‑related content. Real estate: key to retention and safety.

Resources hub

UI element: PCOS 101, diet tips, doctor prep cards. Why: education is foundational. Value: gives practical steps for users and clinicians. Real estate: rounds out the all‑in‑one promise.

Bottom navigation

UI element: Five‑tab bar with active state. Why: matches mobile mental model. Value: makes the hub feel complete. Real estate: always‑available key sections.
